# Sequential Circuits in Verilog

This repository contains structured implementations of fundamental and advanced sequential digital circuits designed using Verilog HDL. It focuses on understanding state-based logic, timing behavior, and real-world hardware modeling.

## Overview

Sequential circuits are digital systems whose outputs depend on current inputs and past states. This repository demonstrates how memory elements like flip-flops and registers drive system behavior in clock-driven environments.

## Covered Concepts

* Latches and Flip-Flops (SR, D, JK, T)
* Registers (SISO, SIPO, PISO, PIPO)
* Counters (Asynchronous & Synchronous)
* Shift Registers
* Sequence Detectors
* Finite State Machines (Moore & Mealy models)

## Tools Used

* Vivado 

## Key Learning Outcomes

* Designing clock-driven systems
* Understanding state transitions and timing
* Building FSM-based logic systems
* Writing synthesizable RTL code
* Debugging using waveform analysis
